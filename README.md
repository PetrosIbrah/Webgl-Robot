## 🤖 WebGL Robot
A 3D interactive robot built entirely with WebGL and JavaScript, rendered directly in the browser. The robot is composed of what looks like cube-based geometry (but is in its depths, triagle geometry) with texture mapping, a skybox, and a flat floor — all controllable through an interactive UI.


## 💻 Demonstration
Open `Index.html` with a Live Server to see the robot in action.

## 🚀 How to Use 
### Prerequisites
- A modern browser (Chrome, Firefox, Edge)
- [VS Code](https://code.visualstudio.com/) with the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) *(recommended)* | Or any IDE alongside a Live Server
### Running the project

1. Clone the repository:
   ```bash
   git clone https://github.com/PetrosIbrah/Webgl-Robot.git
   ```
2. Open the folder in VS Code or Any IDE
3. Right-click `Index.html` and select **"Open with Live Server"**.
4. The project will open in your browser at `http://127.0.0.1:5500`.
